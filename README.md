# Sistema de Gerenciamento de Estoque 📦

## 💻 Sobre o Projeto
Este projeto é uma aplicação desenvolvida em Python para realizar o controle de entrada, saída e saldo de produtos. Ele foi construído com foco na aplicação prática de estruturas de dados, lógica de programação e persistência de dados.

O sistema possui integração direta com um banco de dados SQL, garantindo que todas as movimentações fiquem salvas e organizadas de forma segura e eficiente.

## ⚙️ Funcionalidades
- Registro de entrada de novos produtos no estoque.
- Registro de saída/baixa de produtos.
- Consulta de saldo atualizado em tempo real.
- Armazenamento e persistência de dados utilizando banco de dados.

## 🛠️ Tecnologias Utilizadas
- **Linguagem:** Python
- **Banco de Dados:** SQL
- **Conceitos aplicados:** Lógica de programação, manipulação de dados, estruturação e integração com banco de dados.

## 🚀 Como executar o projeto

1. Clone este repositório:
`git clone https://github.com/paulacervone/sistema-gerenciamento-estoque.git`
2. Acesse a pasta do projeto:
`cd sistema-gerenciamento-estoque`
3. Execute o arquivo principal:
`python main.py`

*(Nota: Se houver a necessidade de instalar bibliotecas externas, adicione o comando `pip install -r requirements.txt` nesta seção).*
